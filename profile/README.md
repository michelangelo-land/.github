# Michelangelo

Michelangelo is an AI-powered iOS app for vibe-coding native **Expo** and **React Native** apps. Build, preview, and share your ideas directly from your iPhone using natural language. [Read the documentation](https://docs.michelangelo.land) to get started.

## Who's using Michelangelo?

Michelangelo is used by indie makers, developers, and product teams to turn ideas into real apps in minutes.

## What makes Michelangelo different?

Michelangelo combines advanced AI, automatic code generation, and deep integration with popular development ecosystems like **GitHub** and **Expo**. Whether you're prototyping an idea or shipping a full-stack app, Michelangelo turns natural language into working code.

| Capability         | Description                                              |
| ------------------ | -------------------------------------------------------- |
| Code generation    | Create native mobile code from natural language          |
| Best practices     | Generate production-ready, best-practice code            |
| Instant viewing    | Preview your app instantly as you build                  |
| Share app          | Share your app with anyone via a simple public link      |
| GitHub integration | Automatically connect and push to GitHub repositories    |
